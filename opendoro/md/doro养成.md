## 一、系统总览

### 世界观

Doro 居住在一个温暖的小镇上，这里有水果店、便利店、公园、咖啡馆、图书馆等。Doro 可以出门探索、打工赚钱、购物消费、结交朋友。核心货币是 **🍊 欧润吉**。

### 核心循环

```
┌─────────────────────────────────────────────────────┐
│                   Doro 的日常                          │
│                                                       │
│   🍅 欧润吉钟专注                                      │
│       ↓                                               │
│   获得 🍊 欧润吉                                       │
│       ↓                                               │
│   🏪 购物 / 🍳 烹饪 / 💼 打工                         │
│       ↓                                               │
│   获得物品 / 更多🍊 / 新食材                            │
│       ↓                                               │
│   🍖 投喂 / 🎮 玩耍 / 🧹 清洁 / 😴 休息               │
│       ↓                                               │
│   Doro 属性提升 → 触发事件 → 等级成长                   │
│       ↓                                               │
│   解锁新区域 / 新物品 / 新打工                          │
└─────────────────────────────────────────────────────┘
```

### 养成维度

| 维度 | 说明 |
|------|------|
| **属性养成** | 饱食度 / 心情值 / 清洁度 / 能量值（已有） |
| **经济养成** | 欧润吉赚取与消费（已有基础） |
| **等级养成** | Doro 等级 Lv.1~10（已有） |
| **物品收集** | 食物、玩具、装饰、食材、收藏品 |
| **关系养成** | 与 NPC 的好感度 |
| **小屋养成** | 家居装饰与布置 |
| **探索养成** | 解锁新地点、触发事件 |

```
       │               │               │

  │ ☕ 咖啡馆│   │  📚 图书馆  │  │  🎪 游乐场 │
  └─────────┘   └────────────┘  └───────────┘
```

### 3.2 地点详情

#### 🏠 Doro家（默认）
- **功能**：查看属性、互动、出发探索
- **特殊**：小屋装修在此进行
- **现有状态**：已实现为 `PetStatusInterface`

#### 🍊 欧润吉水果店
- **功能**：Doro 可以在这里打工赚取欧润吉
- **打工内容**：整理水果、招呼客人、搬运欧润吉
- **收益**：3~8 🍊 /次（根据 Doro 能量值浮动）
- **特殊**：每天可以免费领一个欧润吉（如果饱食度 < 30）
- **解锁**：Doro Lv.2

#### 🏪 便利店
- **功能**：购买各种物品
- **商品分类**：
  - 🍖 食物类（便当、零食、饮料）
  - 🎮 玩具类（球、毛线团、拼图）
  - 🧹 日用品（毛巾、香皂、小扫帚）
  - 🎀 装饰品（蝴蝶结、小帽子、围巾）
  - 🍳 食材类（面粉、鸡蛋、黄油）
- **解锁**：Doro Lv.1（初始可用）

#### 🌳 公园
- **功能**：散步、触发随机事件、遇到 NPC
- **散步收益**：心情 +10，能量 -5
- **随机事件概率**：40%
- **特殊**：天气晴朗时事件概率翻倍
- **解锁**：Doro Lv.1（初始可用）

#### ☕ 猫咪咖啡馆
- **功能**：Doro 打工、社交、喝饮料
- **打工内容**：当"猫店员"招待客人
- **收益**：5~12 🍊 /次
- **特殊**：打工后心情 +5（被客人夸了）
- **解锁**：Doro Lv.4

#### 📚 镇图书馆
- **功能**：Doro 学习新技能、阅读故事
- **技能学习**：消耗欧润吉解锁新互动方式
- **特殊**：每周可以读一本书，提升 Doro 智力（影响 Galgame 对话质量）
- **解锁**：Doro Lv.3

#### 🎪 游乐场
- **功能**：Doro 游玩、触发欢乐事件
- **项目**：旋转木马（🍊×3）、摩天轮（🍊×5）
- **收益**：心情 +20~30
- **特殊**：有小概率获得限定玩具
- **解锁**：Doro Lv.5

### 3.3 外出机制

Doro 外出需要消耗 **1 点行动力**。每天默认有 **3 点行动力**，凌晨 0:00 刷新。

| 行为 | 消耗行动力 | 说明 |
|------|----------|------|
| 散步（公园） | 1 | 30min，可能触发事件 |
| 打工（水果店/咖啡馆） | 1 | 1h，固定收益 |
| 购物（便利店） | 0 | 不耗行动力 |
| 游乐场 | 1 | 1h，心情大幅提升 |
| 图书馆 | 1 | 1h，学习技能 |

外出期间 Doro 从桌面"走出去"，回来时带回收益。可以通过 **欧润吉钟专注** 加速时间流逝。

### 6.2 物品属性

每个物品实体包含：

```python
@dataclass
class InventoryItem:
    id: int              # 唯一ID
    item_key: str        # 物品标识 (如 "orange_jelly")
    name: str            # 名称 "欧润吉果冻"
    category: str        # 分类 "food"
    quantity: int        # 堆叠数量
    effects: dict        # {"hunger": 15, "mood": 5}
    rarity: str          # "common" / "rare" / "legendary"
    description: str     # 物品描述
    obtained_at: str     # 获得时间
```

## 十三、📱 数据存储设计

### 13.1 新增数据库表（pet.db 扩展）

```sql
-- 物品栏
CREATE TABLE inventory (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    item_key TEXT NOT NULL,
    name TEXT NOT NULL,
    category TEXT NOT NULL,
    quantity INTEGER DEFAULT 1,
    rarity TEXT DEFAULT 'common',
    obtained_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 打工记录
CREATE TABLE work_records (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    location TEXT NOT NULL,
    job_name TEXT NOT NULL,
    earnings INTEGER NOT NULL,
    worked_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- NPC 好感度
CREATE TABLE npc_affection (
    npc_id TEXT PRIMARY KEY,
    affection INTEGER DEFAULT 0,
    last_interaction TIMESTAMP
);

-- 小屋家具
CREATE TABLE house_furniture (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    furniture_key TEXT NOT NULL,
    zone TEXT NOT NULL,
    placed_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 成就
CREATE TABLE achievements (
    achievement_key TEXT PRIMARY KEY,
    unlocked_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 已学技能
CREATE TABLE learned_skills (
    skill_key TEXT PRIMARY KEY,
    learned_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 收藏品
CREATE TABLE collectibles (
    collectible_key TEXT PRIMARY KEY,
    obtained_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 每日/每周任务进度
CREATE TABLE quest_progress (
    quest_key TEXT PRIMARY KEY,
    progress INTEGER DEFAULT 0,
    target INTEGER NOT NULL,
    completed INTEGER DEFAULT 0,
    date TEXT NOT NULL
);
```
