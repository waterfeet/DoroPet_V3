## 🎯 项目简介

**DoroPet** 是一款革命性的桌面宠物应用，它不仅仅是一个会动的桌宠，更是你的智能工作伙伴！

想象一下：当你独自面对屏幕工作时，有一个可爱的 Live2D 角色陪伴着你——它会追逐你的鼠标、在屏幕边缘探头探脑、在你疲惫时提醒你休息，甚至能和你进行智能对话、一起玩 Galgame 互动故事、还能为你播放音乐！

### 🌟 为什么选择 DoroPet？

| 特性                 | 描述                                                  |
| ------------------ | --------------------------------------------------- |
| 🎭 **Live2D 动态角色** | 流畅的 Live2D v3 模型渲染，支持表情切换、动作播放、鼠标追踪、冒泡对话            |
| 🤖 **多模型 AI 对话**   | 支持 OpenAI、DeepSeek、Claude、Gemini、Ollama 等 8 种 AI 模型 |
| 🎙️ **语音交互**       | 唤醒词检测 + 语音识别 + 语音合成，让对话更自然                          |
| 🎮 **养成系统**        | 饥饿、心情、清洁度、能量四大属性，属性联动，打造专属养成体验                      |
| 📖 **Galgame 叙事**  | AI 驱动的互动叙事游戏，好感度系统、多结局、存档读档、背包商店                    |
| 🎵 **在线音乐**        | 多平台音乐搜索播放、歌词频谱、歌单管理、VLC 引擎                          |
| 🔌 **Agent 技能**    | 内置 11+ 实用技能，文档处理/搜索/天气/网页抓取/前端设计等，支持自定义扩展           |
| 🧠 **智能记忆**        | AI 自动提取长期记忆，分析消息重要性，让对话更连贯                          |
| 🎨 **主题切换**        | 明暗双主题，支持字体缩放，随心切换                                   |
| 📌 **边缘吸附**        | 宠物可吸附到屏幕四边，自动隐藏/探出                                  |
| 📎 **附件处理**        | 支持 txt，html，json，md 等文件自动提取文本发送给 AI                 |
| 🔄 **自动更新**        | 内置版本管理，启动自动检测，一键更新到最新版本                             |

## 🚀 快速开始

### 📋 系统要求

- **操作系统**: Windows 10/11 (64位)
- **Python**: 3.12+（仅源码安装需要）
- **内存**: 4GB RAM 以上
- **存储空间**: 500MB 以上可用空间
- **显卡**: 支持 OpenGL 3.0+

### 🔧 安装步骤

#### 方式一：下载发行版（推荐）

1. **下载最新版本**

   前往 [Releases 页面](https://gitee.com/waterfeet/DoroPet_V3/releases) 下载最新版本的压缩包
2. **解压文件**

   将下载的 ZIP 文件解压到任意目录（路径中请避免中文和特殊字符）
3. **运行安装脚本**

   双击运行 `install_env.bat`，脚本将自动完成以下操作：
   - 下载 Python 3.12 嵌入式版本
   - 安装 pip 包管理器
   - 安装所有依赖项
   - 配置运行环境
4. **启动应用**

   安装完成后会自动启动，或双击 `start_app.bat` 手动启动

#### 方式二：源码安装（开发者）

适合想要参与开发或自定义修改的用户。

1. **确保已安装 Python 3.12+**
2. **克隆项目**
   ```bash
   git clone https://gitee.com/waterfeet/DoroPet_V3.git
   cd DoroPet_V3
   ```
3. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```
4. **启动应用**
   ```bash
   python main.py
   ```

### ⚡ 首次运行

首次启动后，你需要：

1. **配置 AI 模型** - 进入「模型配置」页面，添加你的 API Key
2. **选择模型** - 在下拉菜单中选择要使用的 AI 模型
3. **开始对话** - 点击桌宠或进入「AI 聊天」页面开始互动

### 🎮 养成系统

四大核心属性，打造沉浸式养成体验：

| 属性         | 说明        | 影响          |
| ---------- | --------- | ----------- |
| 🍖 **饱食度** | 定时消耗，需要投喂 | 过低加速心情衰减    |
| 😊 **心情值** | 通过互动提升    | 影响宠物反应与语录   |
| 🛁 **清洁度** | 定时下降      | 需要清洁保持      |
| ⚡ **能量值**  | 追逐/溜达消耗   | 休息恢复，影响活动能力 |

**陪伴玩法**：

| 类别     | 操作           | 效果        |
| ------ | ------------ | --------- |
| **投喂** | 零食 / 正餐 / 大餐 | 恢复饱食度     |
| **玩耍** | 轻度 / 愉快 / 剧烈 | 提升心情，消耗能量 |
| **清洁** | 擦拭 / 洗澡      | 恢复清洁度     |
| **休息** | 小憩 / 沉睡      | 恢复能量      |
| **互动** | 抚摸 / 责备 / 安慰 | 影响心情      |

> 💡 四大属性之间存在**联动机制**：低饱食度会加速心情衰减，低心情值会延缓能量恢复等，培养时需要统筹兼顾。

3. 选择提供商类型（OpenAI / DeepSeek / Claude / Gemini / Kimi / Zhipu / Groq / Ollama）
4. 填写 API Key 和相关配置（Base URL、模型名等）
5. 保存并激活

### 🎮 开始 Galgame

1. 进入「Galgame」页面
2. 点击顶部配置面板设置世界观、主角、角色
3. 点击「开始新游戏」或「继续上次进度」
4. AI 会实时生成剧情，在出现选项时做出你的选择
5. 每个选择都会影响故事走向、角色好感度，最终导向不同结局

> 💡 Galgame 中的好感度、货币、道具会持久化保存，可在「背包」中查看道具，在「好感度」中查看各角色好感。

| **AI 接口**     | OpenAI SDK (兼容多提供商)，Provider 适配层架构            |
| **语音处理**      | sherpa-onnx (唤醒词+ASR) + edge-tts / OpenAI TTS |

| **音乐引擎**      | VLC + musicdl (多平台音乐搜索)                       |
| **代码高亮**      | Pygments                                      |

### 扩展开发

#### 添加新的 AI 提供商

1. 在 `src/provider/sources/` 创建新的提供商文件
2. 继承 `LLMProvider` 基类
3. 实现必要的方法（`chat`、`stream_chat` 等）
4. 在 `src/provider/register.py` 中注册提供商元数据

#### 添加新技能

1. 在 `src/skills/` 创建技能目录
2. 编写 `SKILL.md` 配置文件（遵循技能规范）
3. 添加执行脚本（可选，用于复杂技能）
4. 重启程序自动加载，或在「技能管理」页面刷新

#### 开发插件

1. 在 `plugin/` 目录下创建新文件夹
2. 创建 `main.py`，其中包含一个继承自 `QWidget` 的 `Plugin` 类
3. 重启程序，插件会自动出现在「插件管理」页面

## ❓ 常见问题

<details>
<summary><b>Q: 安装环境失败？</b></summary>

A: 请检查以下几点：

- **路径问题**：确保安装路径中不包含中文、空格或特殊字符（如 `D:\软件\DoroPet` ❌ → `D:\DoroPet` ✅）
- 网络连接是否正常
- 是否被杀毒软件或防火墙拦截
- 尝试以管理员身份运行安装脚本

</details>

<details>
<summary><b>Q: 下载依赖时提示网络错误？</b></summary>

A: 脚本内置多个镜像源（清华源/阿里源/中科大源等），如果全部失败，请检查：

- 网络连接是否正常
- 是否被防火墙拦截
- 尝试使用 VPN 或更换网络环境

</details>

<details>
<summary><b>Q: Live2D 模型显示异常？</b></summary>

A: 请确保：

- 显卡驱动已更新
- 支持 OpenGL 3.0+
- 模型文件完整未损坏（需包含 `.model3.json` 配置文件）

</details>

<details>
<summary><b>Q: AI 对话无响应？</b></summary>

A: 请检查：

- API Key 是否正确配置
- 网络是否能访问对应 API 地址
- Base URL 是否填写正确（使用自定义代理时）
- 查看「运行日志」页面获取详细错误信息

</details>

<details>
<summary><b>Q: 如何添加自定义 Live2D 模型？</b></summary>

A: 将模型文件夹放入 `models/` 目录，确保包含 `.model3.json` 配置文件及相关的表情/动作/物理文件，然后在「Live2D模型」页面选择加载。

</details>

<details>
<summary><b>Q: 语音唤醒不灵敏？</b></summary>

A: 语音唤醒使用本地 sherpa-onnx 模型，建议：

- 在相对安静的环境中使用
- 清晰说出唤醒词 "Hey Doro"
- 确认麦克风权限正常
- 可在「语音设置」中调整相关参数

</details>

<details>
<summary><b>Q: 如何使用 Ollama 本地模型？</b></summary>

A:

1. 在本地安装 [Ollama](https://ollama.com/) 并拉取模型
2. 在「模型配置」中选择 Ollama 提供商
3. Base URL 填写 `http://localhost:11434/v1`（默认地址）
4. 模型名填写你在 Ollama 中拉取的模型名称（如 `qwen2.5:7b`）

</details>

<details>
<summary><b>Q: Galgame 存档丢失？</b></summary>

A: Galgame 存档存储在 SQLite 数据库中，保存在应用数据目录下。正常情况下不会丢失。如希望备份，可在「通用设置」中查看数据目录路径，手动备份相关数据文件。

</details>
